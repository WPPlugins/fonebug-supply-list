A simple, but effective list-making tool for you to keep track of items that you have and items that you need.

== Description ==

Fonebug Supply List for WordPress lets you create an unlimited number of lists and add individual items to each list.

You can check off the items you currently have or don't have and order the list alphabetically or by items you don't have.

== Installation ==
1. Download and unzip the Fonebug Supply List folder.
2. Copy the "fonebug-supply-list" folder to the WordPress directory: "/wp-content/plugins/". NOTE! - do not copy the folder with the version number.
3. Upload the "fonebug-supply-list" folder to your server.
4. Log in to your WordPress admin page.
5. Click "Plugins" from your admin menu.
6. Find "Fonebug Supply List" on your list of plugins.
7. Click Activate.
8. A "Supplies" menu item will appear on your admin menu.
9. Click "Supplies" from your admin menu.
10. Fonebug will automatically build the database tables required to run the plugin.
11. After the tables are built, your Administration page will load.
12. Follow the prompts at the top of the Admin page to get started building your schedule. 
15. Once installation is complete, you can delete the file: "_include/fb-db-build.php". But you don't have to.
16. If you have any issues or questions, PLEASE leave a support message and I will do my best to help you.
